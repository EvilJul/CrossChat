## 🚀 如何应用新图标

### 方法 1: 自动生成（推荐）

1. **转换 SVG 为 PNG**
   ```
   访问: https://svgtopng.com/
   上传: design/app-icon.svg
   尺寸: 1024 x 1024
   下载: 保存为 icon-1024.png
   ```

2. **生成所有尺寸**
   ```bash
   cargo tauri icon design/icon-1024.png
   ```

3. **重新编译**
   ```bash
   npm run tauri build
   ```

### 方法 2: 手动替换

如果你已经有 PNG 文件，可以手动替换 `src-tauri/icons/` 目录中的文件。

## 📐 技术规格

### 图标尺寸要求

**Windows**:
- 16×16, 32×32, 48×48, 64×64, 128×128, 256×256

**macOS**:
- 16×16, 32×32, 64×64, 128×128, 256×256, 512×512, 1024×1024

**Linux**:
- 32×32, 128×128, 256×256, 512×512

**Windows Store**:
- 30×30, 44×44, 71×71, 89×89, 107×107, 142×142, 150×150, 284×284, 310×310

### 设计规范

- **圆角半径**: 22.5% (512px 图标为 115px)
- **安全区域**: 中心 80% 区域
- **色彩空间**: sRGB
- **格式**: PNG (透明背景), ICO, ICNS

## 🔧 故障排除

### 问题: Tauri Icon 命令找不到

**解决**:
```bash
cargo install tauri-cli
```

### 问题: 图标模糊

**原因**: 源文件分辨率太低  
**解决**: 使用 1024×1024 或更高分辨率的 PNG

### 问题: 小尺寸图标不清晰

**原因**: 细节过多  
**解决**: 对 128×128 以下使用 `app-icon-simple.svg`

### 问题: Windows 图标显示异常

**解决**: 删除 `src-tauri/icons/` 目录，重新运行 `cargo tauri icon`
