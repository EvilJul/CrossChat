📝 如何使用这些图标
快速步骤：
1. 访问 svgtopng.com
2. 上传 app-icon.svg，设置尺寸为 1024×1024
3. 下载 PNG，保存为 icon-1024.png
4. 运行: cargo tauri icon design/icon-1024.png
5. 重新编译: npm run tauri build
