# CrossChat 图标设计文件

## 🚀 快速开始

### 最简单的方法（推荐）

1. **在线转换 SVG 为 PNG**
   - 访问: https://svgtopng.com/
   - 上传 `app-icon.svg`
   - 设置尺寸: **1024 x 1024**
   - 下载 PNG，保存为 `icon-1024.png`

2. **使用 Tauri 生成所有尺寸**
   ```bash
   # 在项目根目录运行
   cargo tauri icon design/icon-1024.png
   ```

3. **重新编译应用**
   ```bash
   npm run tauri build
   ```

完成！✨
